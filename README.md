**项目用到 Vue2.6 + vue-router3.1.2 + echarts4.9.0重构完啦！**

![](./src/assets/image/1635048227.jpg)

## 前端主要技术

- vue: ^2.6.12
- axios: ^0.21.0
- echarts: ^4.9.0
- sass: ^1.29.0

## Build Setup

```bash
# clone the project
git clone https://github.com/Zore-One/vue2-echarts.git

# enter the project directory
cd vue2-echarts

# install dependency
npm install

# develop
npm run serve
```
